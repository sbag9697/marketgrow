# 🚀 CoolSMS 5분 설정 가이드

## 1. CoolSMS 가입 (1분)

1. https://coolsms.co.kr 접속
2. **회원가입** 클릭
3. 이메일로 간단 가입 (사업자등록증 불필요)
4. 이메일 인증 완료

## 2. API 키 발급 (1분)

1. 로그인 후 **대시보드** 이동
2. 왼쪽 메뉴에서 **API Key 관리** 클릭
3. **새 API Key 생성** 클릭
4. API Key와 API Secret 복사 (한 번만 표시됨!)

## 3. 발신번호 등록 (2분)

1. **발신번호 관리** 메뉴 클릭
2. **발신번호 등록** 클릭
3. 본인 휴대폰 번호 입력
4. 인증번호 받아서 입력
5. **등록 완료**

## 4. 무료 포인트 확인

- 가입 시 **무료 30포인트** 자동 지급
- SMS 1건 = 1포인트
- 30건 테스트 가능!

## 5. Railway 환경변수 설정

Railway 대시보드 → Variables에 추가:

```
COOLSMS_API_KEY=복사한_API_KEY
COOLSMS_API_SECRET=복사한_API_SECRET  
COOLSMS_SENDER=01012345678  (등록한 발신번호)
```

## 6. 패키지 설치 (선택사항)

로컬 테스트 시에만 필요:

```bash
cd backend
npm install coolsms-node-sdk
```

Railway는 자동으로 설치됨!

## 7. 테스트 방법

회원가입 페이지에서:
1. 전화번호 입력
2. **인증** 버튼 클릭
3. SMS 수신 확인
4. 인증번호 입력

## 💰 요금 충전 (필요시)

### 충전 방법
1. **포인트 충전** 메뉴
2. 금액 선택 (최소 1,000원)
3. 결제 진행

### 요금표
- SMS (90자): 20원/건
- LMS (2000자): 50원/건
- MMS (이미지): 100원/건

## 🔧 문제 해결

### "잔액 부족" 오류
→ 무료 포인트 소진, 충전 필요

### "발신번호 미등록" 오류
→ 발신번호 관리에서 번호 등록

### "인증 실패" 오류
→ API Key 확인, 환경변수 확인

## 📱 개발 모드로 테스트

SMS 설정 없이도 테스트 가능:

1. Railway 환경변수 설정 안 함
2. 인증 요청 시 콘솔에 코드 출력
3. 개발/테스트용으로 충분
