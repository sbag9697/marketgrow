# 📞 카카오톡 채널 고객센터 정보 설정 방법

## 문제 해결: "고객센터 정보가 입력되어 있는지 확인해주세요"

### 1. 올바른 경로 찾기

카카오톡 채널 관리자센터의 고객센터 정보는 여러 곳에 숨어있을 수 있습니다:

## 방법 1: 채널 관리자센터 (신규 UI)

1. https://business.kakao.com 로그인
2. **카카오톡 채널** 클릭
3. 해당 채널 선택
4. 왼쪽 메뉴에서 **관리** → **채널 정보**
5. **기본 정보** 섹션에서:
   - 운영시간
   - 전화번호
   - 이메일
   - 홈페이지 URL
   입력

## 방법 2: 비즈니스 정보 설정

1. **관리** → **비즈니스 정보**
2. 다음 정보 입력:
   ```
   상호명: MarketGrow (또는 사업자명)
   대표자명: [대표자 이름]
   사업자등록번호: [사업자번호]
   통신판매업 신고번호: (있으면 입력)
   고객센터 전화번호: 010-xxxx-xxxx
   고객센터 이메일: support@marketgrow.kr
   고객센터 운영시간: 평일 09:00~18:00
   주소: [사업장 주소]
   ```

## 방법 3: 프로필 편집에서 설정

1. **프로필** → **프로필 편집**
2. 스크롤을 내려서 **추가 정보** 섹션
3. **+ 정보 추가** 클릭
4. 선택 가능한 항목:
   - 📞 전화번호
   - 📧 이메일
   - 🏠 주소
   - ⏰ 운영시간
   - 🌐 웹사이트

## 방법 4: 홈 설정에서 추가

1. **홈** → **홈 관리**
2. **정보** 탭 클릭
3. **기본 정보 설정** 또는 **정보 관리**
4. 다음 항목 추가:
   - 연락처 (전화번호)
   - 운영 시간
   - 위치/주소
   - 홈페이지

## 자주 놓치는 부분

### ⚠️ 필수 입력 항목
```
✅ 전화번호 (고객센터 또는 대표번호)
✅ 운영시간 (예: 평일 09:00~18:00)
✅ 이메일 주소
✅ 홈페이지 URL (https://marketgrow.kr)
```

### 💡 팁: 모바일 앱에서 설정
카카오톡 채널 관리자 앱을 사용하면 더 쉽게 찾을 수 있습니다:

1. **카카오톡 채널 관리자** 앱 다운로드
2. 로그인
3. **설정** → **채널 정보**
4. **고객센터 정보** 섹션

## 임시 해결책

고객센터 정보 입력란을 찾을 수 없다면:

### 1. 채널 소개에 직접 입력
```
📞 고객센터: 010-xxxx-xxxx
📧 이메일: support@marketgrow.kr
⏰ 운영시간: 평일 09:00~18:00
💬 카카오톡 1:1 채팅 가능
```

### 2. 자동응답 메시지에 포함
```
안녕하세요! MarketGrow입니다 😊

📞 고객센터: 010-xxxx-xxxx
⏰ 운영시간: 평일 09:00~18:00
📧 문의: support@marketgrow.kr

무엇을 도와드릴까요?
```

### 3. 홈 포스트에 게시
1. **홈** → **포스트 작성**
2. "고객센터 안내" 포스트 작성
3. 상단 고정

## 카카오 고객센터 문의

위 방법으로도 해결되지 않는다면:

### 카카오톡 채널 고객센터
- 전화: 1544-4293
- 카카오톡: @카카오톡채널 (1:1 채팅)
- 이메일: partner@kakaocorp.com

### 문의 내용 예시
```
안녕하세요.
카카오톡 채널(@marketgrow) 운영자입니다.
고객센터 정보 입력란을 찾을 수 없어 문의드립니다.
채널 관리자센터에서 정확한 위치를 안내해주세요.
```

## 대안: 프로필 링크 활용

고객센터 정보를 별도 페이지로 만들고 링크:

1. **프로필** → **링크 추가**
2. 링크 제목: "고객센터"
3. URL: https://marketgrow.kr/support
4. 설명: "문의 및 상담"
