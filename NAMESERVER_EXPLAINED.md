# 🌐 네임서버란? (초보자용 쉬운 설명)

## 📱 전화번호부 같은 것!

네임서버는 **인터넷의 전화번호부**입니다.

### 예시로 이해하기:
```
사람이 이해하는 주소: marketgrow.com
컴퓨터가 이해하는 주소: 192.168.1.1 (IP 주소)

네임서버 = 이 둘을 연결해주는 번역기 📖
```

## 🏠 일상생활 비유

1. **도메인** = 집 주소 (서울시 강남구 테헤란로 123)
2. **IP 주소** = GPS 좌표 (37.5665, 126.9780)
3. **네임서버** = 네비게이션 (주소를 좌표로 변환)

누군가 "marketgrow.com"을 입력하면:
1. 네임서버가 "아, 그건 75.2.60.5에 있어요!" 라고 알려줌
2. 브라우저가 그 IP 주소로 연결
3. 웹사이트가 나타남!

## 🔧 네임서버 설정이 필요한 이유

### 현재 상황:
- **도메인**: 가비아에서 구매 (marketgrow.com)
- **웹사이트**: Netlify에서 호스팅
- **문제**: 가비아는 marketgrow.com이 어디 있는지 모름!

### 해결:
네임서버를 Netlify 것으로 바꿔주면 됨!

## 📝 네임서버 변경 방법 (가비아)

### Step 1: 가비아 로그인
1. https://www.gabia.com 접속
2. My가비아 → 도메인 관리

### Step 2: 네임서버 변경
1. 구매한 도메인 옆 [관리] 클릭
2. [네임서버 설정] 클릭
3. "타사 네임서버" 선택

### Step 3: Netlify 네임서버 입력
```
1차: dns1.p01.nsone.net
2차: dns2.p01.nsone.net
3차: dns3.p01.nsone.net
4차: dns4.p01.nsone.net
```

### Step 4: 저장
- [적용] 버튼 클릭
- 10분~1시간 후 적용 완료!

## 🎯 네임서버 vs DNS 레코드

### 네임서버 변경 (쉬운 방법) ✅
- 모든 DNS 관리를 Netlify에 맡김
- 한 번만 설정하면 끝
- **초보자 추천!**

### DNS 레코드 설정 (복잡한 방법)
- 가비아에서 직접 관리
- A 레코드, CNAME 등 개별 설정
- 전문가용

## 🚦 네임서버 변경 후 일어나는 일

```
[변경 전]
marketgrow.com → 가비아 네임서버 → ??? (연결 안됨)

[변경 후]
marketgrow.com → Netlify 네임서버 → Netlify 서버 → 웹사이트 표시!
```

## ⏰ 적용 시간

| 지역 | 적용 시간 |
|------|-----------|
| 한국 | 10분~1시간 |
| 아시아 | 1~6시간 |
| 전세계 | 최대 48시간 |

확인 방법: https://dnschecker.org 에서 도메인 입력

## 💡 자주 묻는 질문

### Q: 네임서버를 바꾸면 이메일도 영향받나요?
A: 네, 이메일 설정도 Netlify에서 해야 합니다.

### Q: 네임서버 변경을 취소할 수 있나요?
A: 네, 언제든 원래대로 되돌릴 수 있습니다.

### Q: 네임서버가 여러 개인 이유는?
A: 백업용입니다. 하나가 고장나도 다른 것이 작동합니다.

## 🎮 쉬운 비유 정리

네임서버는 **인터넷 주소록**입니다!

```
전화번호부: 홍길동 → 010-1234-5678
네임서버: marketgrow.com → 75.2.60.5

전화번호부 회사를 바꾸는 것 = 네임서버 변경
가비아 전화번호부 → Netlify 전화번호부로 교체!
```
