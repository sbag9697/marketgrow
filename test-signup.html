회원가입 API 테스트

문제 해결 체크리스트

  • 콘솔(F12)에서 CORS 오류 확인
  • 네트워크 탭에서 실제 요청 확인
  • 응답 상태 코드 확인

직접 CURL 테스트

curl -X POST https://marketgrow-production.up.railway.app/api/auth/register \
  -H "Content-Type: application/json" \
  -d '{"username":"test999","email":"test999@example.com","password":"Test123!@#","name":"Test","phone":"01099998888"}'
